title:

hexo + node +git 搭建个人博客


date: 2017-12-26 16:34:59
tags:
reward: true

一、环境、工具的安装

1、node：一款web服务器，目前很多大型项目都使用node搭建

1. 特点：解析效率高，没有兼容性；比较适合I/O密集型的项目
2. 版本：简易版本是6.10（比较稳定，兼容性好）点击
链接 相应安装包，按步骤安装直至最后。在命令行中输入
node -v 若显示版本，则安装成功，否则重装。

2、vscode：多语言开发工具

3、git 版本控制器

二、安装hexo脚手架

1、在命令行输入 npm install -g hexo-cli.

1. npm: 依赖包管理工具
2. -g: global 全局安装

2、初始化一个博客项目。

hexo init 项目名称（英文）

四、进入添加依赖包

1、进入blog文件夹（项目主目录文件夹），鼠标右键选择git bash here

2、使用npm install 命令添加依赖包

目的： 将项目所需依赖包添加到本地

五、启动服务器：

1、在博客目录右键git bash here

2、输入hexo server

3、在浏览器地址栏输入：localhost：4000 ，然后点击Enter

六、发表文章

hexo new 文章名

七、博客预览

1、ctrl+s 保存

2、hexo g 编译

3、回到浏览器ctrl+r 刷新
