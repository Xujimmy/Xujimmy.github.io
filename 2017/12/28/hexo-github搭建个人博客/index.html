hexo+github搭建个人博客

hexo 是目前最主流的静态博客框架，该框架基于node，因此编译效率非常高，同时借助于github可以实现零成本搭建与维护。

环境搭建

安装node环境

在node官网下载对应操作系统的node安装包，安装过程比较简单，就是next模式。安装完成后通过以下步骤检测是否安装成功。

  • win+r打开命令行窗口
  • 输入cmd回车
  • 在cmd窗口输入node -v

如果提示”node不是外部或内部命令”，则表示安装失败。

安装git

git:版本控制器。
进入git官网下载对应操作系统的git安装包，next模式安装。

安装编译器

vscode是一款强大的开发工具(IDE)

安装hexo脚手架

  1. win+r输入cmd回车

  2. 输入npm install hexo-cli -g 回车

    • npm: 依赖包管理器。
    • install： 安装、导入，可以简写为 i
    • -g： 全局安装

清理缓存

当修改了内容，编译后浏览器没有发生变化时，需要清理blog缓存。方式如下：

  1. 在blog目录执行hexo clean后重新编译
  2. 在浏览器博客页面使用ctrl+shift+i（f12) 调出控制台，切换到network，选择disable cashe， 然后ctrl+r（刷新）

替换主题

  1. 在主题官网找到自己喜欢的主题，根据提示git到本地themes目录
  2. 修改博客项目,根据路径下的_config.yml配置文件，将theme改为新主题文件夹名称。
  3. 停止hexo服务在重启（ctrl+c）
  4. 执行hexo g 编译，然后去浏览器预览。

托管到github

  1. 注册github账号（百度）
  2. 验证ssh key
    • 在桌面右键 git bash here.
    • 使用 ssh-keygen -t rsa -C “注册git的邮箱”命令，然后一路回车，直到出现泡沫图案。
    • 进入C:\User\计算机用户名.ssh 文件夹
    • 右键打开id_rsa.pub文件，然后ctrl+a全选，然后ctrl+c复制
    • 进入github官网，右上角点击个人头像，下拉菜单选择settings
    • 选择左侧ssh and gpg keys
    • 单击右侧add ssh keys
    • title： 随便填写，粘贴前面复制的keys，然后点击 add key。

    • 仓库名： github用户名.github.io
      ps：注意区分大小写，必须保持一致。

个人域名绑定

  1. 域名申请（百度）
  2. 域名解析，添加如下记录值：
    记录类型|主机记录|解析线路(isp)|记录值
    -|-|-|-
    CNAME|www|默认|用户名.github.io
    A|@|默认|192.30.352.153
    A|@|默认|192.30.352.154

  4. 在CNAME文件里面填写个人域名，如baidu.com
    ps: 不带http、www等修饰。
